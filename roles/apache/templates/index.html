La página del Proyecto

En la página de GitHub encontrarás todos los archivos y roles de Ansible para la creación de los servidores web, incluyendo uno que servirá para gestionar los demás.

Las máquinas virtuales las podrás crear mediante un Vagrantfile que usará Virtualbox. Puedes ajustar los nombres de las máquinas o la memoria ram desde este archivo.

Si necesitas saber más sobre cómo funcionan las diferentes máquinas o cómo configurarlas tras ejecutar los roles, entra a la página del proyecto de GitHub.

El nodo gestor

Es una máquina virtual con Debian 11 que se actualiza automáticamente e instala la última versión de Anisble junto con Python3. Usa esta para ejecutar roles de Ansible en el resto de servidores.

El servidor de Grafana

Grafana nos permite conectarnos a diversos servidores web para proporcionarnos datos de manera gráfica; un ejemplo de su uso sería la monitorización de recursos del servidor, pero en este proyecto hemos decidido que lo vamos a usar para ver los datos del servidor de PostgreSQL.

La página de Wordpress

Hemos decidido añadir un Wordpress para que tenga la función de "blog" dentro de este entorno de máquinas virtuales. La página en la que estás leyendo esto, que está funcionando con Apache, simplemente se encarga de informar y redirigir.

En caso de que estas máquinas se fuesen a usar en servidores web con una función real, como para una pequeña empresa, podría usar esta página como principal para los clientes y redirigirlos a las diferentes secciones, y usar el WordPress como foro de noticias, por ejemplo.

El servidor de PostgreSQL

Es un servidor de base de datos que está corriendo en CentOS7. Puede acceder al mismo conectándote mediante ssh a la máquina, y luego a la base de datos, o directamente a la base de datos mediante el acceso remoto del propio Postgres.
